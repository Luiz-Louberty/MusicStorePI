# MusicStore - Sistema de Gestão de Inventário
**Status:** Em desenvolvimento (Etapa 5)

**Objetivo:** Sistema desktop para gerenciar estoque e vendas de uma loja de instrumentos musicais.

**Tecnologias:**
- Java JDK 17 (Swing)
- JPA / Hibernate (Persistência)
- MySQL (Banco de Dados)
- Maven (Dependências)

**Funcionalidades:**
- Cadastro completo de instrumentos (CRUD).
- Registro de vendas vinculadas ao CPF do cliente.
- Persistência automática no banco de dados.
